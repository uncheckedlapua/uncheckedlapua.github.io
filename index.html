Summary: Latitude vs. X

The purpose of this project was to analyze how weather changes as you get closer to the equator. With data pulled from the a dataset on over 500 locations.


Factors we looked at include: temperature, cloudiness, wind speed, and humidity. This site provides the source data and visualizations created as part of the analysis.
